HTML TAG LIST

SI Tag Names Descriptions
01 <!DOCKTYPE> Defines the document type
02 <a> Defines a hyperlink
03 <abbr> Defines an abbreviation or an acronym
04 <address> Defines contact information for the author/owner of a document
05 <area> Defines an area inside am image-map
06 <article> Defines an article
07 <b> Defines Bold Text
08 <body> Defines the document's body
09 <br> Defines a single line break
10 <code> Defines a piece of computer code
11 <del> Defines text that has been deleted from a document
12 <div> Defines a section in a document
13 <dl> Defines a description list
14 <dt> Defines a term/name in a description list
15 <h1>to<h2> Defines HTML headings
16 <head> Defines information about the document
17 <html> Defines the root of an HTML document
18 <i> Defines a part of text in an altrnate voice or mood
19 <iframe> Defines an inline inframe
20 <img> Defines an image
21 <ins> Defines a text that has been inserted into a document
22 <input> Defines an input control
23 <link> Defines the relationship between a document and an external resource(most used to link to style sheets)
24 <li> Defines a list item
25 <map> Defines a client-side image-map
26 <p> Defines a paragraph
27 <strong> Defines important text
28 <style> style informatiom for a document
29 <table> Defines a table
30 <td> Defines a cell in a table
